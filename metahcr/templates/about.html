About MetaHCR

If you use MetaHCR for any published research, please include the following citation: MetaHCR: a web-enabled metagenome data management system for hydrocarbon resources" Peter C. Marks, Marc Bigler, Eric B. Alsop, Bart P. Lomans and Nicolas Tsesmetzis (2017)

MetaHCR Milestones

  • Development of a database containing microbial species and functional genes identified in the literature or the analyzed samples
  • Implementation of a cyber-infrastructure to be used across the project for data storage, analysis and sharing
  • Creation of a user-friendly front-end allowing easy access to the data by the non-expert
